# Aether Chat

A real-time chat application with AI-powered chat features.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm start
```

3. Open the app in your browser:

http://localhost:3000

## Notes

- The server listens on the port defined by `PORT` in `.env`, or defaults to `3000`.
- If you want to change the port, create a `.env` file with:

```env
PORT=4000
```

- This project uses Express and WebSocket for real-time messaging.
