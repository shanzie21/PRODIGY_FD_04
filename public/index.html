Security Policies

Account Lockout Protocol

To protect your account against unauthorized brute-force attempts, Aether Chat implements strict security lockouts:

  • Failed Attempts Allowed: 8 consecutive incorrect logins.
  • Lockout Duration: 15 minutes for the first lockout tier.
  • Active Logging: Every session contains the connecting User Agent and IP address. Revoking a session immediately drops their WebSocket connections and forces a logout.
